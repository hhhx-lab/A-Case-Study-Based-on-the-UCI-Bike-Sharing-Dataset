# 07 统计推断检验方案

## 1. 本部分目标

探索性图表可以展示差异，但不能单独说明差异是否具有统计意义。因此，论文需要加入统计推断检验。统计推断部分可以体现 R 在经典统计分析中的应用，也能让论文更像正式的数据分析报告。

本部分要回答：

1. 工作日与非工作日的租赁量是否存在显著差异？
2. 不同季节的租赁量是否存在显著差异？
3. 不同天气状况下的租赁量是否存在显著差异？
4. 2011 年与 2012 年的租赁量是否存在显著差异？
5. 对右偏分布或非正态情况，应如何使用稳健检验？

## 2. 总体原则

统计检验不应机械堆叠，而应围绕研究问题设计。

原则：

1. 两组比较使用 t 检验，并用 Wilcoxon 秩和检验作为稳健补充。
2. 多组比较使用单因素方差分析，并用 Kruskal-Wallis 检验作为稳健补充。
3. 如果多组差异显著，再做事后比较。
4. 不只报告 p 值，还应报告均值、中位数或组间差异方向。
5. 样本量很大时，p 值容易显著，因此要结合实际差异解释。

## 3. 工作日与非工作日差异检验

### 3.1 研究假设

原假设：

工作日与非工作日的平均总租赁量没有显著差异。

备择假设：

工作日与非工作日的平均总租赁量存在显著差异。

### 3.2 R 代码

```r
working_summary <- bike_hour |>
  group_by(workingday) |>
  summarise(
    n = n(),
    mean_cnt = mean(cnt),
    median_cnt = median(cnt),
    sd_cnt = sd(cnt),
    .groups = "drop"
  )

t_working <- t.test(cnt ~ workingday, data = bike_hour)
w_working <- wilcox.test(cnt ~ workingday, data = bike_hour)

working_test_table <- tibble(
  test = c("t 检验", "Wilcoxon 秩和检验"),
  statistic = c(unname(t_working$statistic), unname(w_working$statistic)),
  p_value = c(t_working$p.value, w_working$p.value)
)
```

### 3.3 写作解释

不能只写“p < 0.05，所以显著”。建议写：

> 本文首先比较工作日与非工作日的总租赁量差异。若 t 检验和 Wilcoxon 秩和检验均显示 p 值小于 0.05，则说明两类日期的租赁量分布存在统计显著差异。结合前文小时曲线可进一步发现，这种差异并不只是平均水平差异，更体现在工作日早晚高峰更明显，而非工作日白天需求更平缓。

注意：

工作日均值初步为 193.21，非工作日均值初步为 181.41，差异不一定非常大。因此要强调“模式差异”比“均值差异”更重要。

## 4. 季节差异检验

### 4.1 研究假设

原假设：

四个季节的平均租赁量相同。

备择假设：

至少有一个季节的平均租赁量不同。

### 4.2 R 代码

```r
season_summary <- bike_hour |>
  group_by(season) |>
  summarise(
    n = n(),
    mean_cnt = mean(cnt),
    median_cnt = median(cnt),
    sd_cnt = sd(cnt),
    .groups = "drop"
  )

anova_season <- aov(cnt ~ season, data = bike_hour)
anova_season_table <- broom::tidy(anova_season)

kw_season <- kruskal.test(cnt ~ season, data = bike_hour)

tukey_season <- TukeyHSD(anova_season)
```

### 4.3 写作解释

已知季节均值：

| 季节 | 平均租赁量 |
| --- | ---: |
| 春季 | 111.11 |
| 夏季 | 208.34 |
| 秋季 | 236.02 |
| 冬季 | 198.87 |

写作示例：

> 从分组均值看，春季平均租赁量较低，秋季平均租赁量较高。进一步通过单因素方差分析检验季节差异，如果检验结果显著，则说明季节因素与共享单车租赁量存在统计关联。考虑到租赁量分布右偏，本文同时使用 Kruskal-Wallis 检验作为非参数稳健检验。

## 5. 天气状况差异检验

### 5.1 研究假设

原假设：

不同天气状况下的平均租赁量相同。

备择假设：

不同天气状况下的租赁量存在差异。

### 5.2 R 代码

```r
weather_summary <- bike_hour |>
  group_by(weathersit) |>
  summarise(
    n = n(),
    mean_cnt = mean(cnt),
    median_cnt = median(cnt),
    sd_cnt = sd(cnt),
    .groups = "drop"
  )

anova_weather <- aov(cnt ~ weathersit, data = bike_hour)
kw_weather <- kruskal.test(cnt ~ weathersit, data = bike_hour)
```

### 5.3 特殊注意

`weathersit = 恶劣天气` 只有 3 条记录。统计检验时可以保留，但解释时必须说明样本太少。

更稳健的做法：

```r
bike_weather3 <- bike_hour |>
  filter(weathersit != "恶劣天气")

anova_weather3 <- aov(cnt ~ weathersit, data = bike_weather3)
kw_weather3 <- kruskal.test(cnt ~ weathersit, data = bike_weather3)
```

论文写法：

> 由于恶劣天气类别仅包含极少数观测，本文在描述统计中保留该类别，但在稳健性检验中进一步剔除该类别，比较晴朗或少云、多云或有雾、小雨雪三类天气下的租赁量差异。

这会体现你分析很细。

## 6. 年度差异检验

### 6.1 研究假设

原假设：

2011 年和 2012 年的平均租赁量没有显著差异。

备择假设：

2011 年和 2012 年的平均租赁量存在显著差异。

已知初步均值：

- 2011 年：143.79
- 2012 年：234.67

### 6.2 R 代码

```r
year_summary <- bike_hour |>
  group_by(yr) |>
  summarise(
    n = n(),
    mean_cnt = mean(cnt),
    median_cnt = median(cnt),
    sd_cnt = sd(cnt),
    .groups = "drop"
  )

t_year <- t.test(cnt ~ yr, data = bike_hour)
w_year <- wilcox.test(cnt ~ yr, data = bike_hour)
```

### 6.3 写作解释

> 年度比较可以反映共享单车系统在两年间的总体需求变化。如果 2012 年平均租赁量显著高于 2011 年，可能说明共享单车系统使用规模扩大或用户接受度提升。但由于数据中没有直接提供投放车辆数量、用户注册规模等运营信息，因此本文只将年度变量解释为时间趋势控制变量，不做严格因果推断。

## 7. 相关性检验

可以对连续环境变量与租赁量做 Pearson 和 Spearman 相关分析。

```r
cor_vars <- bike_hour |>
  select(cnt, temp_celsius, atemp_celsius, hum_percent, windspeed_real)

cor_pearson <- cor(cor_vars, method = "pearson")
cor_spearman <- cor(cor_vars, method = "spearman")
```

解释：

1. Pearson 反映线性相关。
2. Spearman 反映单调关系，对非正态和极端值更稳健。
3. 相关分析不能控制其他变量，因此只能作为回归前的初步分析。

## 9. 统计语言规范

推荐表述：

- “检验结果显示，组间差异具有统计显著性。”
- “在 5% 显著性水平下拒绝原假设。”
- “该结果说明不同季节的租赁量分布存在差异。”
- “考虑到样本量较大，本文不只依据 p 值判断，还结合分组均值和可视化结果进行解释。”

避免表述：

- “p 值显著证明天气导致租赁量下降。”
- “季节决定租赁量。”
- “模型完全正确。”
- “差异一定来自某某原因。”

## 10. 原创性与查重提醒

统计检验部分原创性来自：

1. 自己计算的分组均值和检验结果。
2. 自己选择的检验方法组合。
3. 对 p 值和实际差异的谨慎解释。

不要复制教材中对 t 检验、ANOVA 的大段定义。只需用一两句话说明为什么使用该方法。论文重点应放在本数据结果，而不是方法百科。
